# 📚 Repositorio de Prácticas SQL  

Este repositorio contiene consultas SQL y anotaciones teóricas para reforzar el aprendizaje de bases de datos. Incluye ejemplos prácticos de consultas, optimización y uso de diferentes tipos de comandos SQL.  

## 📌 Contenido  

- 📖 **Teoría SQL**: Explicaciones sobre comandos y conceptos clave.  
- 📝 **Consultas SQL**: Ejercicios y ejemplos de consultas para practicar.  
- 🔍 **Optimización**: Estrategias para mejorar el rendimiento de las consultas.  

## 🔹 Temas Cubiertos  

✔️ **DDL (Data Definition Language)**  
- `CREATE`, `ALTER`, `DROP`  
- Claves primarias y foráneas  

✔️ **DML (Data Manipulation Language)**  
- `SELECT`, `INSERT`, `UPDATE`, `DELETE`  
- Uso de `JOIN`, `GROUP BY`, `HAVING`  

✔️ **DCL (Data Control Language)**  
- `GRANT`, `REVOKE`  

✔️ **TCL (Transaction Control Language)**  
- `COMMIT`, `ROLLBACK`, `SAVEPOINT`  

✔️ **Funciones y Procedimientos Almacenados**  
- Creación de funciones (`CREATE FUNCTION`)  
- Procedimientos almacenados (`CREATE PROCEDURE`)  

✔️ **Optimización y Buenas Prácticas**  
- Índices (`INDEX`)  
- Normalización de bases de datos  
- Subconsultas y CTEs (`WITH`)
